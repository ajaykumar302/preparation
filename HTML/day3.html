1. Difference between flex-direction row and column. Ans- flex-direction: row - It is the sub property of flex which is the property of CSS used to display items on horizontal axis in row manner. We can reverse the row order in flex. Syntax: div { display: flex; flex-direction: row; } flex-direction: column- It is the sub property of flex which is the property of CSS used to display items on vertical axis in column manner. Similar to row, we can reverse the column order in flex. Syntax: div { display: flex; flex-direction: column; } 2. Explain inline, internal, and external stylesheet.
